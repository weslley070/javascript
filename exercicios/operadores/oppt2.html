Toda expressão que é calculada por operadores relacionais retorna valores booleanos, ou seja, true or false

5 e "5" são iguais se calculados por == que calcula identidade.
Já o operador === verifica se os doi valores comparados são IDÊNTICOS tanto em tipo quanto em valor.

Operaddores lógicos

Vou usar o exemplo das canetas:

! negação: Não quero uma caneta vermelha.Pode ser qualquer caneta menos vermelha
&& conjunção: Quero uma caneta vermelha e azul.Tem que ser uma azul e uma vermelha
|| disjunção: Quero uma caneta vermelha ou azul.Pode ser uma vermelha ou uma azul

Um operador lógico pode ser binario ou unario
unario é apenas o ! que so retorna true e false
ja o && e || retornam dois valores sejam eles true or false
Para o && retornar true as dua expressões comparadas devem ser true.Qualquer outro resultado retorna false
Ja o || para que ele retorne true basta uma das expressões ser verdadeira.Se as duas expressões forem falsas ele retorna false.

ordem de precedencia dos operadores:
1 aritimeticos 2 relacionais 3 logicos

Operador ternario

Ele realiza um teste logico no começo e de acordo com o resultado ele faz o que você pediu.Exemplo:
Media >= 7 ? 'aprovado' : 'reprovado'
O teste logico é feito no inicio, e se o resultado for true ele escreve o que esta no primeiro bloco.Se for falso ele ecreve o que esta no segundo.
formula: Teste logico ? bloco com o que vai ser escrito se o resultado for true : bloco com o que vai ser escrito se o valor for false.

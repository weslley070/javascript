operadores aritimeticos

Ordem de precedencia

1 () 2 ** 3 */% 4 +-

+ para adição

- para subtração

* para multiplicação

/ para divisão

% para o resto da divisão inteira

** para potenciação

se você quiser que certa operação seja feita primeiro você pode usar ()

Operadores de auto atribuições

var n = 3
n = n + 4 e tambem posso usar todos os operadores aritimeticos
pode tambem se utilizar n+= 4 para simmplificar, mas apenas se a variavel receber ela mais 4

Operadores de incremento / decremento

Existem dois tipos de incremento e decremento:
o pré e o pós.
o pré é feito assim: ++var ou para subtração --var
Já o pós é feito assim: var++ ou var--
